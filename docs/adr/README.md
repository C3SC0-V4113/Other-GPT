# ADR Index

Este directorio contiene Architecture Decision Records (ADR) del proyecto.

## Convencion

- Formato de nombre: `NNNN-short-title.md`
- Numeracion secuencial de 4 digitos:
  - `0001-current-architecture.md`
  - `0002-...`
  - `0003-...`
- Estado sugerido: `Proposed`, `Accepted`, `Superseded`, `Deprecated`.

## Flujo recomendado

1. Registrar contexto y problema.
2. Documentar decision y alternativas consideradas.
3. Describir consecuencias y tradeoffs.
4. Referenciar impacto en `README.md`, `DESIGN.md` y `AGENTS.md` cuando aplique.
